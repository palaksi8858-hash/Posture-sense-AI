# Posture-sense-AI
AI-based posture detection system

## 📌 Overview
Posture Sense AI is an AI-based posture detection system designed to monitor and correct body posture in real-time. It helps prevent health issues like neck pain, back pain, and fatigue caused by prolonged sitting and poor posture.

## 🚀 Features
- Real-time posture detection using camera
- Alerts for incorrect posture (slouching, neck bending)
- Improves focus and productivity
- Helps build long-term healthy sitting habits

## 🧠 How It Works
The system uses computer vision techniques to track body posture through a camera.  
It analyzes key body points and detects whether the user is sitting correctly or not.  
If incorrect posture is detected, it provides instant feedback to correct it.

## 💻 Tech Stack
- HTML / CSS / JavaScript  
- AI Logic (Computer Vision)  
- Assisted by ChatGPT & Claude for development  

## 🎯 Use Cases
- Students (online classes)
- Office workers (long sitting hours)
- Gamers
- Healthcare (basic posture monitoring)
